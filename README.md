# webhook

processing webhooks easy

## Supported platforms

- [x] [Resend](https://resend.com/docs/dashboard/webhooks/introduction)
  - Implemented event types
    - [x] `email.sent`
    - [x] `email.delivered`
    - [x] `email.delivery_delayed`
    - [x] `email.complained`
    - [x] `email.bounced`
    - [x] `email.opened`
    - [x] `email.clicked`
    - [x] `contact.created`
    - [x] `contact.updated`
    - [x] `contact.deleted`
